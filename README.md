# rESEM using SCAD
## Project Overview
Research in social and behavioral sciences increasingly relies on high-dimensional data, creating challenges for traditional Structural Equation Modeling approaches. Therefore new research into models that can deal with this data is needed. This paper provides a continuation to this research by extending a newly proposed Exploratory Structural Equation Model. To test this new model a simulation study is setup. Results show that the proposed model achieves comparable performance when tested against the existing models. ## Usage
This repository contains 4 files, three of them are used for running the model and the other one is used for generating the plots. For generating the data take a look at: https://github.com/trale97/regularizedESEM. To run the model simpely run the rESEM_SCAD_sim.R file, do not forget to change the input, output folders and sources to the other files first.
